# LUMA Programming Language

## About

**LUMA Programming Language** is an academic project developed as part of a programming languages course. 
The project demonstrates the design and implementation of a custom programming language using lexical analysis, parsing, and interpretation techniques.

## Developers

- Bea Sandara Abonales
- Friensan Q. Pablo

## Technologies Used

- Lex (Lexical Analyzer)
- Yacc (Parser Generator)
- Python

## Project Description

LUMA is a custom programming language created to explore the fundamental concepts behind compiler and interpreter construction. The project uses **Lex** to perform lexical analysis, **Yacc** to generate the parser, and **Python** to implement the language's supporting functionality.

## Disclaimer

This repository contains an **academic project** developed for educational purposes. It is intended to demonstrate the concepts of programming language design and implementation and is not intended for production use.
